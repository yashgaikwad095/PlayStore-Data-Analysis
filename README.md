# Retail Store Sales Analytics 

## Overview
This project demonstrates an end-to-end relational database analysis of a retail store's sales and inventory data using **MySQL Workbench**. The objective was to extract actionable business insights regarding sales trends, product performance, and customer purchasing patterns.

## Tech Stack Used
* **Database Management System:** MySQL
* **Tools:** MySQL Workbench
* **Core Concepts:** Data Definition Language (DDL), Data Manipulation Language (DML), Joins, Aggregation Functions, Subqueries

## Key Insights & Operations
* Designed and queried complex relational tables to manage retail data.
* Executed multi-table `JOIN` operations to connect sales records with product inventory.
* Utilized aggregation functions (`SUM`, `AVG`, `GROUP BY`) to identify top-performing product categories and high-value customer segments.
* Maintained clean, optimized SQL scripts for accurate financial reporting.

## How to Run
1. Download the `.sql` script provided in this repository.
2. Open MySQL Workbench and connect to your local server.
3. Open the script and execute the queries to view the database structure and analysis results.
